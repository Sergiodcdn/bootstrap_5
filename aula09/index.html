Class lead + img. Por padrão a imagem fica no start e texto contarna ela, dentro das classes foi alterado a ordem, assim como a justificativa do texto

A classe "lead" é uma classe de estilo do Bootstrap que é usada para dar destaque a um parágrafo de texto, tornando-o mais proeminente e fácil de ler. É comumente usada para o primeiro parágrafo de um bloco de texto, como um resumo ou introdução, e pode ser aplicada a qualquer elemento < p> ou < div> no HTML. A classe "lead" aplica um tamanho de fonte maior e um espaçamento entre linhas aumentado, o que ajuda a destacar o texto e torná-lo mais legível. Além disso, ela também adiciona uma margem inferior ao elemento, o que ajuda a separá-lo visualmente de outros elementos de texto na página.
