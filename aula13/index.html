Margens Externas e Internas

Externas = margin, internas = padding

"rem" é uma unidade de medida que significa "root em", ou seja, é relativa ao tamanho da fonte raiz do documento HTML (normalmente definida como 16 pixels).

Código Efeito
m/p-0 remove toda a margem do elemento
m/p-1 adiciona uma margem pequena ao elemento (0,25 rem)
m/p-2 adiciona uma margem média ao elemento (0,5 rem)
m/p-3 adiciona uma margem grande ao elemento (1 rem)
m/p-4 adiciona uma margem muito grande ao elemento (1,5 rem)
m/p-5 adiciona uma margem enorme ao elemento (3 rem)

m - s(start)/e(end)/t(top)/b(bottom). Esses sufixos deixam configurar de forma individual os espaçamentos.

p - x(horizontal)/y(vertical)

Todas as classes tem variações baseadas nos breakpoints de responsividade.
